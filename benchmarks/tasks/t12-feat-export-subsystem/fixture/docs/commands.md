# orderdesk commands

Run any command as `node src/cli.js <command> [options]`.

| Command | Description |
|---|---|
| list-products | Print every product as `<id>  <name>  $<price>` |
| list-orders | Print every order as `<id>  <customerId>  $<total>` |
| list-customers | Print every customer as `<id>  <name>  <email>` |
| add-product | Add a product: `add-product <id> <name> <price>` |
| help | Print all available commands |
